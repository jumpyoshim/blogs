# 目的

HTTP の雰囲気理解を脱する

# 要約

HTTP は WWW におけるデータの転送方法で、開始行、メッセージヘッダ、ボディで構成される。Cookie によってセッションステートレス問題を解消する。また、HTTPS は HTTP にはない暗号化・認証・改ざん防止機能をもつ。HTTPS は公開鍵暗号方式を使用しデジタル証明書によってサーバー認証を行う。
